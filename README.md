# forma-website
Packaging Design Studio Website


## Hero showcase

Первый экран теперь использует композицию:
- одна большая главная упаковка спереди;
- остальные упаковки находятся позади;
- каждые 4.6 секунды главный продукт меняется;
- поддерживается до 8 PNG;
- отдельный фон у композиции отсутствует.

PNG-файлы меняются в:
`assets/js/hero-packs.js`
